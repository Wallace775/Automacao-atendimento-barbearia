# 🤖 Automação de Atendimento - Barbearia Premium

Projeto de chatbot inteligente desenvolvido para otimizar o fluxo de agendamentos e suporte ao cliente, reduzindo o tempo de resposta e automatizando processos repetitivos.

### 🛠️ Tecnologias e Ferramentas
* **Typebot**: Engenharia de fluxo e lógica conversacional.
* **HTML5/CSS3**: Integração customizada via Iframe no portfólio.
* **JavaScript**: Scripts de controle de interface e carregamento.

### 🧠 Lógica do Fluxo
O bot foi estruturado para guiar o usuário em três frentes principais:
1. **Consulta de Serviços**: Exibição dinâmica de cortes e valores.
2. **Agendamento**: Direcionamento para horários disponíveis.
3. **Suporte**: Transbordo para atendimento humano via WhatsApp.
